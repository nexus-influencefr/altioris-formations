# Images nécessaires

Veuillez ajouter les images suivantes dans ce dossier :

## Images obligatoires

1. **`logo-altioris.png`**
   - Logo Altioris Formation
   - Format : PNG avec transparence recommandé
   - Hauteur recommandée : 50px pour le header
   - Formats acceptés : PNG, SVG, JPG

2. **`logo-qualiopi.png`**
   - Logo QUALIOPI officiel (si disponible)
   - Format : PNG recommandé
   - Taille recommandée : ~120px de largeur pour la section qualité
   - Formats acceptés : PNG, SVG, JPG

3. **`hero-bg.jpg`**
   - Image de fond pour la hero section de la page d'accueil
   - Format : JPG recommandé
   - Dimensions recommandées : 1920x1080px (ou plus)
   - Ratio : 16:9 ou similaire
   - Poids : Optimisé pour le web (< 500KB)

## Images optionnelles

4. **`formation-placeholder.jpg`**
   - Image par défaut pour les formations
   - Dimensions recommandées : 800x600px
   - Ou créer des images spécifiques pour chaque formation

5. **`team-placeholder.jpg`**
   - Photo par défaut pour les membres de l'équipe
   - Format carré : 600x600px
   - Ou photos spécifiques pour chaque membre

## Notes

- Toutes les images doivent être optimisées pour le web
- Utiliser des formats modernes (WebP si possible)
- Respecter les droits d'auteur
- Ajouter les attributs `alt` appropriés dans le HTML après ajout des images
