# Documents nécessaires

Veuillez ajouter les documents PDF suivants dans ce dossier :

## Documents réglementaires (obligatoires)

1. **`qualiopi-certification.pdf`**
   - Certificat de certification QUALIOPI de l'organisme
   - Format : PDF
   - Ce document est référencé sur la page Documents et doit être accessible en téléchargement

2. **`accessibilite.pdf`**
   - Déclaration de conformité d'accessibilité numérique
   - Format : PDF
   - Obligatoire pour les sites publics et certains organismes

3. **`cgv.pdf`**
   - Conditions Générales de Vente des formations
   - Format : PDF
   - Obligatoire pour tout organisme de formation

4. **`mentions-legales.pdf`**
   - Mentions légales et identification de l'organisme
   - Format : PDF
   - Obligatoire

## Documents complémentaires

5. **`statuts.pdf`** (si applicable)
   - Statuts de l'organisme
   - Format : PDF

6. Autres documents spécifiques à votre organisme :
   - Programme de formation
   - Contrat de formation
   - Règlement intérieur
   - etc.

## Notes

- Tous les documents doivent être au format PDF
- Vérifier que les liens de téléchargement dans `documents.html` correspondent aux noms de fichiers
- Les documents doivent être à jour et conformes à la réglementation en vigueur
- Respecter le RGPD pour tout document contenant des données personnelles

## Liens dans le HTML

Les documents sont référencés dans `documents.html` avec des liens de téléchargement. 
Vérifier que les chemins correspondent aux fichiers que vous ajoutez.

Example :
```html
<a href="documents/qualiopi-certification.pdf" download>Télécharger</a>
```
